Wow yeah, that really did go well. For months I've been planning on setting up my website with Netlify and Gatsby.js and there was going to be a whole learning curve that my main line of work takes priority over.

However today I decided to slap on a theme onto my github pages website and boom. Then looking at Jekyll themes in more detail it turns out you can just set up the theme in your git repo by cloning from a master repo and bam, website up and running with blog posts and everything.

It's incredibly easy and now I look forward to working with the markdown and page structure. Plus it's great not having to deal with a database and server.

*I guess I just need to do some proper styling and write some actual content now...*
