Project Description

Pemberly Fox is a service for bespoke stationery. When I took the website on it needed serious work on correcting features such as the product filtering. It appeared that the website had a version upgrade instead of a straight forward patch. This caused some custom modules (in this case Manadev Layered Navigation) to cease functioning.

In addition to cleaning up some of the styling of the website, one of my main tasks was to improve the optimisation of the website. Using GTMetrix as the baseline we took the website from an E&F score up to an A&B score with the A score achieving 100% as of this writing
